用Vercel免费托管你的静态网页(可自定义域名).md

发表于 2022-08-13 更新于 2024-08-18 分类于 技术分享 本文字数： 532 阅读时长 ≈ 1 分钟

来源：https://www.zsuh.cc/post/e14b15f6.html

前期准备

  1. 一个漂漂亮亮的源码
  2. 一个 Github 账户
  3. 一个你的个性域名（非必须，vercel 也会为你提供一个免费域名）
  4. 一个智慧的大脑（没有请按 ctrl+w）

操作步骤

下载并解压我们的源码，按照自己的需求进行修改（￣～￣）

（注：如果不会修改的话可以找我，我可以浅教一下）

下面我们打开 Github，并且新建一个项目，项目名任意，这里我新建的项目名是我自己的 github 名字 inwinter04
网址: https://github.com/

新建完成后按照图中的步骤打开上传界面，并且拖入我们的源码文件，点击绿色的那个提交按钮。


现在源码已经上传到我们的 Github 仓库了

下面请出我们今天的主角 ——vercel。
网址: https://vercel.com/

点击注册，选择 Github 登录并且授权。
（注：因为我已经注册过了，所以这里就不演示怎么注册。有看不懂英语请使用网页翻译）
（注：网上有看到过使用 QQ 或网易邮箱可能会出现未知问题。该消息并未被证实，并且我使用的也是网易邮箱，暂时没有发现任何问题。）

登录完成，我们开始导入项目。选择我们刚刚在 Github 里面新建的那个仓库，也就是我们上传源码的那个项目。
